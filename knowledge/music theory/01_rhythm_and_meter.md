# Rhythm and Meter

## Core idea

Rhythm is not “notes in time.”
Rhythm is **hierarchy in time**:
- strong vs weak beats
- division vs subdivision
- expectation vs surprise
- repetition vs variation

## What the student must understand

- pulse
- beat
- meter
- subdivision
- duration
- accent
- syncopation
- groove

## Why it matters for composition

Weak writing often comes from:
- no pulse hierarchy
- no contrast between strong and weak positions
- every event having equal importance
- too many events and not enough space

Strong writing usually has:
- a clear pulse
- clear placement of accents
- controlled density
- some contrast between stable and unstable placements

## Hearing model

### Stable placements
Events on metrically strong positions feel grounded.

### Unstable placements
Events on weak positions feel lighter or more directional.

### Syncopation
A syncopated event gains energy because it lands where the listener expects less emphasis.

## Composition heuristics

- First make the pulse obvious
- Then vary subdivisions
- Then remove notes to create groove
- Then displace one event to create tension

## Core drills

### Drill 1 — Strong beats
Write a one-bar pattern with events only on strong beats.

Goal:
hear stability

### Drill 2 — Subdivision contrast
Write the same idea using:
- quarter notes
- eighth notes
- sixteenth notes

Goal:
hear density and urgency change

### Drill 3 — Syncopation
Take a stable rhythm and move one event to a weaker position.

Goal:
hear extra forward motion

### Drill 4 — Space as structure
Delete one expected event from a repeated pattern.

Goal:
hear how silence shapes groove

## Contrast prompts

Ask:
- which version feels more grounded?
- which version feels more urgent?
- which version feels like it leans forward?
- which version breathes more?

## Common mistakes

- filling every available slot
- confusing complexity with groove
- ignoring accent hierarchy
- changing too many variables at once
