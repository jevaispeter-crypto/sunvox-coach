# Intervals and Melody

## Core idea

Melody is not a list of notes.
Melody is **motion with memory**.

Intervals determine:
- contour
- tension
- singability
- stability
- expressive character

## What the student must understand

- step vs leap
- repetition
- contour
- interval quality
- tension and release within a line
- phrase direction

## Useful intuition

### Stepwise motion
Feels smooth, connected, singable.

### Leaps
Feel more noticeable and dramatic.
Large leaps create energy, but need balance.

### Repetition
Creates identity.

### Contour
Upward motion often feels like growth or lift.
Downward motion often feels like settling or release.

## Practical melody rules

- start simple
- use more steps than leaps
- repeat and vary a small cell
- if you leap, often compensate later
- aim for a phrase peak and a phrase release

## Core drills

### Drill 1 — Stepwise melody
Write a 5-note phrase using only stepwise motion.

Goal:
hear smoothness and coherence

### Drill 2 — One leap only
Write a 5-note phrase with exactly one leap larger than a third.

Goal:
hear how one leap changes shape

### Drill 3 — Motive variation
Write 3 notes.
Repeat them once exactly.
Then repeat them a third time with one note changed.

Goal:
hear identity + variation

### Drill 4 — Question and answer
Write one 4-note phrase that feels open.
Write a second 4-note phrase that feels more closed.

Goal:
hear phrase direction

## Contrast prompts

Ask:
- which version is easier to sing?
- which version feels more dramatic?
- where is the highest-energy moment?
- which ending feels more complete?

## Common mistakes

- random note selection
- too many leaps
- no repeated cell
- no phrase goal
- ending every idea on a strong “home” note
