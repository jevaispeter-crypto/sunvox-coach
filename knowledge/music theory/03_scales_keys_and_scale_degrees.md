# Scales, Keys, and Scale Degrees



\## Core idea

A key is not just a scale.
A key is a **field of gravity**.

Some notes feel:

* stable
* directional
* unresolved
* colorful
* final

Scale degrees matter because they describe function relative to a tonal center.



\## What the student must understand

* tonic as home
* scale-degree identity
* tendency tones
* diatonic collection
* key center
* relative stability of notes



\## High-value scale-degree intuitions

* 1 = home / rest
* 2 = moves easily to 1 or 3
* 3 = colors major/minor identity
* 4 = often leans downward to 3 or participates in predominant motion
* 5 = support / anchor / dominant area
* 6 = softer color; often opens away from closure
* 7 = strongest upward pull to 1 in tonal contexts



\## Why this matters

Without scale-degree awareness:

* melodies feel random
* harmonies feel disconnected
* cadences feel accidental

With scale-degree awareness:

* you can predict expectation
* you can write more intentional motion
* you can hear why endings feel final or incomplete



\## Core drills

### Drill 1 — Tonic hearing

Write a short phrase that ends on 1.
Then write the same phrase ending on 2.

Goal:
hear complete vs open ending

### Drill 2 — Leading tone

Write a phrase ending 7 → 1.

Goal:
hear pull into tonic

### Drill 3 — Third matters

Use the same root and fifth but alternate major vs minor third.

Goal:
hear mode identity

### Drill 4 — Same notes, different tonic

Use the same collection of notes but treat a different pitch as home.

Goal:
hear key-center shift



\## Contrast prompts

Ask:

* which ending feels settled?
* which note sounds like it wants to move?
* which version feels major? minor?
* which pitch sounds like home?



\## Common mistakes

* treating all scale notes as equally stable
* confusing “in the scale” with “functionally identical”
* ignoring the role of 7 → 1 and 4 → 3 tendencies

\## Quick test



If you remove the tonic:

→ does the phrase feel incomplete?



If yes → tonal center is clear
