# Daily Composition Practice

## Core idea

You do not need to “finish masterpieces” every day.
You need consistent contact with:
- hearing
- shaping
- varying
- deciding

## Minimal daily loop (10–20 minutes)

1. Hear one concept
2. Write one short example
3. Write one contrast example
4. Vary one parameter
5. Decide what changed

## Example daily plans

### Day type A — rhythm
- write one stable pattern
- write one syncopated version
- compare groove

### Day type B — melody
- write one 5-note phrase
- write one variation
- compare contour

### Day type C — harmony
- write one progression that resolves
- write one that stays open
- compare closure

### Day type D — texture
- write one 3-layer texture
- remove one layer
- compare clarity

## Weekly minimum targets

- 2 rhythm studies
- 2 melody studies
- 2 harmony studies
- 1 short complete sketch

## Productive constraints

Use limits like:
- only 3 notes
- only 2 chords
- only one rhythmic cell
- only one motive with 3 variations

## Common mistakes

- waiting for inspiration before writing
- always trying to finish full pieces
- never revisiting old ideas
- changing too many variables at once
