# Analysis and Song Mining

## Core idea

One of the fastest ways to improve composition is to identify exactly what you like in real music.

Do not ask only:
- “Do I like this?”

Ask:
- “What, specifically, is working?”

## What to analyze

- rhythm pattern
- melodic cell
- interval profile
- chord loop
- phrase length
- texture roles
- arrangement changes
- cadence type
- energy curve

## Song-mining workflow

1. choose a short excerpt you love
2. identify one thing it does well
3. isolate that thing
4. recreate it simply
5. vary it

## Core drills

### Drill 1 — one-feature extraction
Take one passage and extract only the rhythm.

Goal:
reuse structure without copying everything

### Drill 2 — interval map
Ignore exact notes.
Write only the contour and interval direction.

Goal:
separate shape from pitch surface

### Drill 3 — progression map
Name the harmonic function or root motion.

Goal:
see structure under the sound

### Drill 4 — arrangement map
List each layer and its job.

Goal:
learn role clarity

## Analysis questions

- What repeats?
- What changes first: rhythm, pitch, harmony, texture, or register?
- Where does tension rise?
- Where does release happen?
- What is the smallest useful pattern here?

## Common mistakes

- trying to analyze the entire piece at once
- copying surface details without understanding role
- analyzing passively without rewriting
