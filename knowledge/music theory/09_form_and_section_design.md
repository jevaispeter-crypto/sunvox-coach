# Form and Section Design



\## Core idea

Form is large-scale expectation.

A section should feel like:

* beginning
* continuation
* contrast
* return
* conclusion



\## High-value beginner forms

* period / antecedent-consequent
* sentence
* binary
* ternary
* verse / chorus style contrast



\## What matters more than formal labels

* repetition vs contrast
* pacing
* sectional energy
* where tension rises
* where release occurs



\## Core drills

### Drill 1 — A / A'

Write an 8-unit section.
Repeat it with one clear variation.

Goal:
learn sameness vs difference

### Drill 2 — antecedent / consequent

Write a first phrase ending openly.
Write a second phrase ending more closed.

Goal:
hear question and answer

### Drill 3 — contrast section

Write A, then B with:

* different rhythm
* different register
* different texture

Goal:
hear sectional contrast

### Drill 4 — return

Write A → B → A

Goal:
hear stability of return



\## Contrast prompts

Ask:

* where does the section feel complete?
* does B genuinely contrast A?
* does the return feel earned?



\## Common mistakes

* changing too little between sections
* changing everything at once
* no clear arrival point
* no contrast in texture or rhythm



\## Retrieval tags

form, phrase structure, sentence, period, binary, ternary, section



\## Minimum viable structure (form, not melody)



Use this as a simple composition structure:



A → A' → B → A



Where:

\- A = one short musical idea (can be rhythm, melody, or full pattern)

\- A' = same idea with a small variation

\- B = a contrasting idea (different rhythm, pitch, or texture)

\- A = return to the original idea



This applies to:

\- melody

\- rhythm patterns

\- chord progressions

\- full arrangements in SunVox



Goal:

learn repetition, variation, contrast, and return
