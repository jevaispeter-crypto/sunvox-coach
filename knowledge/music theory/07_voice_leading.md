# Voice Leading

# 

# \## What it is

# 

# Voice leading is the way individual notes (voices) move from one chord to another.

# 

# Instead of thinking only in terms of chords, voice leading focuses on how each note travels over time.

# 

# Good voice leading makes music sound smooth, natural, and intentional.

# 

# 

# \## Core Principles

# 

# \### 1. Prefer small movements

# 

# Notes should move by step (adjacent notes) whenever possible.

# 

# Large jumps make the music feel disconnected.

# 

# 

# \### 2. Keep common tones

# 

# If a note exists in both chords, keep it in the same place instead of moving it.

# 

# Example:

# C major → A minor  

# C stays C

# 

# 

# \### 3. Avoid unnecessary motion

# 

# If nothing needs to change, don’t change it.

# 

# Movement should have a purpose.

# 

# 

# \### 4. Resolve tension properly

# 

# Certain notes create tension and must resolve in a specific way.

# 

# Example:

# \- Leading tone (B in C major) → resolves to C

# 

# 

# \## Why it matters

# 

# Bad voice leading:

# \- sounds jumpy

# \- feels random

# \- breaks musical flow

# 

# Good voice leading:

# \- sounds smooth

# \- feels intentional

# \- connects ideas together

# 

# 

# \## In practice (DAW / SunVox)

# 

# Instead of:

# \- rewriting full chords every time

# 

# Think:

# \- which notes stay

# \- which notes move slightly

# 

# 

# \## Mental model

# 

# Don’t think:

# "I change chord"

# 

# Think:

# "I move voices"
