# Tracker Workflow (Core of SunVox)

\## Pattern System

* Default: 32 lines
* Represents \~2 measures (4/4)

\## Timeline

* Patterns arranged horizontally
* Patterns are independent

\## Note Entry

* Vertical time (top → bottom)
* Each line = time step

\## Edit Step

Controls cursor movement:

* 1 = precise entry
* 2 = useful for eighth notes

\## Note Duration

* Defined by spacing
* NOT by holding key

\## Pattern Independence

Important limitation:

* Notes do NOT continue across patterns

\## Solution

* Extend pattern length
* Or merge patterns

\## Key Principle

Patterns = containers of time  
NOT continuous audio regions



\## Mental Model Shift

DAW mindset:

→ continuous timeline

Tracker mindset:

→ discrete pattern blocks

If you think in DAW terms → you will struggle
