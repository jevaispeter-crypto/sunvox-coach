# App Navigation \& Interface

\## Installation

* Download from warmplace.ru
* Extract and run executable
* Available on desktop and mobile

\## Interface Components

* Module Area → sound routing
* Pattern Editor → sequencing
* Timeline → arrangement
* Piano Keyboard → note input

\## Input Methods

### Computer Keyboard Mapping

* Z = C, X = D, C = E...
* Black keys via S, D, G, etc.

### Octave Control

* Buttons: Octave +/-
* Function keys: F1–F8

\## Modules

Default modules:

* Generator (sound source)
* Echo (effect)
* Output (required)

\## Linking Modules

Two methods:

* Shift + drag
* Link tool

Important:

* Signal flows directionally
* Incorrect direction = no sound

\## Key Principle

No sound → module not connected to Output



Common Mistakes

\- Creating modules but not linking them

\- Linking in wrong direction

\- Forgetting Output module



\## Debug Heuristic

No sound?

1\. Is Generator connected to Output?

2\. Is volume > 0?

3\. Is note actually triggered?
