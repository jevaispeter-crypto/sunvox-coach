# Module System & Sound Design

## What is a Module?

A unit that:
- Generates sound
- Processes sound
- Routes signal

## Types

- Generator (synthesis)
- Sampler (sample playback)
- Effects (echo, filter, etc.)
- Output

## Signal Flow

- Must connect modules manually
- Direction matters

## Key Rule

Source → Effect → Output

## Deleting Modules

- Right-click → Delete
- Backspace

## Loading Modules

- Load from Instruments folder
