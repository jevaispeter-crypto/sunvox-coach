# Composition Workflows

\## Using Sheet Music

Best learning method:

* Import existing scores
* Forces problem solving

\## Pattern Length Strategy

* Short patterns → modular
* Long patterns → continuity

\## Tied Notes

Problem:

* Patterns cut notes

Solution:

* Extend pattern length

\##Repetition

Use:

* Pattern cloning

Benefits:

* Faster workflow
* Consistent edits

\##Triplets

Require:

* Timing adjustments
* Grid or delay effects

\##Creative Insight

Constraints of tracker:
→ force intentional composition

\##Key Principle

Structure first, sound second



\## Workflow Example

1\. Create 1 pattern (32 lines)

2\. Add kick on 0 and 8

3\. Clone pattern

4\. Modify snare

→ builds structure fast
